XLSXGrep


CLI tool to search text in XLSX and XLS files. It works similary to GNU grep.
- Search many XLSX and XLS files at once, even recursively in directories.
- Grep compatible: xlsxgrep tries to be compatible with GNU grep, where it makes sense.
- Some of GNU grep options are supported (such as `-r`, `-i` or `-c`).

Package hosted on PyPI
